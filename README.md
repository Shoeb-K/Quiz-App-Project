# Quiz App

## Project Description

The Quiz App allows users to play a quiz game where individual scores are stored in local storage. At the end of the game, a list of scores is displayed, showing the highest score achieved.

## Tech Stack

- HTML
- CSS
- JavaScript
- IDE: VSCode
